# Low Level Design (LLD) Projects

A collection of **Low Level Design (LLD)** projects implemented in **Java** to demonstrate object-oriented design principles, SOLID principles, and real-world system design. This repository serves as a hands-on resource for learning LLD concepts and preparing for software engineering interviews.

## Current Projects

* **ATM System** – Models the core functionalities of an Automated Teller Machine using a modular and object-oriented design.

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* SOLID Principles
* Design Patterns
